# Bumpy Booby

Bumpy Booby is a simple, responsive and highly customizable PHP bug tracking system. 

**[The demo](https://bumpybooby.herokuapp.com/) is built using this branch.**

Version 1.1.2

## Install

Auto deploy on Heroku.
